# McGemini
Minecraft gemini for interacting with Spigot server
# Getting Started
Install the plugin on your server.
Configure the plugin by editing the config.yml file:
```yml
Config:
  instructions: ""
  apikey: ""
  model: "gemini-1.5-flash-latest"
```
